Claude code 代理開發心得

AI開發時間真的是飛快且沒有什麼大問題 (相較於半年前大概是容易出錯的等級，好可怕)

大部分時間都是在review AI產出的功能，只要功能方向很明確，基本上實現都沒問題，但如果實現的邏輯不夠清晰，就會導致AI產出與你要的不同而來回修改(可能我用sonnet4.6開發的關係?)

Claude Design 使用心得
